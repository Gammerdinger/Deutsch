# Deutsch for English-speakers

The goal of this repository is to create a free, open-source, searchable and sharable resource that people English-speakers learning German can use. Many textbooks that are currently available fall short in two ways:

- Limited exercises
- No explanations in one's mother tongue

The first issue is dictated by the price of paper and development. Sure, a textbook could have 1000 exercise for each concept, but it would make the book much larger and more expensive.

The second reason is also excusable as textbook manufacturers are trying to sell the same textbook in Chile as the United States, so localizing a textbook to each language would be expensive. That's not the goal of this resource. Many of those resources that only focus on the target language exist and I wanted to create a resource specifically for English-speakers who are approaching German. I will try to highlight the parallels and differences between the languages. I acknowledge that there is value in "having to think" in the target language, but oftentimes I feel like a clear explanation in one's mother tongue, particularly for grammar, helps tremendously.

In order to address these concerns, I have created this resource for my own learning and for the learning of others. I will try my best to provide grammar explanations as fully as possible in English, while minimizing the number of exceptions to "rules". 

I am also going to try to embed as many Shiny apps to practice concepts as possible. Please be patient with this as apps and the databases they draw from sometimes take long periods of time to develop. Lessons without apps may exist and that usually means that there is/are an app/s that I am in the processing of developing for the lesson that isn't/aren't ready yet. The goal of the apps is to try to give you near limitless practice of a given concept.

If there are errors, and I expect there will be plenty over time, please open a GitHub issue and I will address it as soon as I can. I really value this feedback as it gives me an opportunity to refine my knowledge as well as the materials.

You are welcome to fork this repository for your own use if you would like to edit it. I hope this resource is helpful for you!
